Tips for writing copy-paste

How to write original copy-paste?

It is difficult to write original copy-paste, like masterpieces of classical literature, but it is possible.

To do this, be creative, think about the problem that hovers around the object, as well as the style of its design.

Unicode characters, which are displayed correctly in almost any browser, will help you bring more emotions into the text

For Twitch chat you can use emotes from BetterTv or FrankerFazez

Technical features

Spaces in ascii

To create pictures based on Braille characters, blank characters, such as the space character, are used to fill in places where dots should be missing.

It's important to note that many messaging services trim extra whitespace, such as extra spaces between words and at the beginning and end of a message. Also, services can simply not display extra whitespace characters by disabling their display via the css white-space property. Because of this, ascii images are displayed incorrectly

To solve this problem, when creating ascii images, you need to use a blank braille character instead of the usual space character.

The following example illustrates this clearly. Try replacing the braille blank character with a regular space, then apply the changes. Whitespace characters will disappear, while Braille characters will remain.

In addition, all braille characters have the same width, unlike blank and alphabetic characters. This is what helps create sufficiently detailed pictures from dots.

New line character

In addition to empty spaces, the following can be removed from messages:

  • newline character;
  • carriage return character;

This is what Twitch does, for example. Therefore, do not forget about line breaks even when the ascii image looks normal.

Design

Before adding copy-paste to the resource, please insert tags. See how the most popular pastes are made.

For Ascii pictures add the ascii tag, as well as a short description in the description field so that you can find the picture using keywords and expressions.

For regular text copy-paste, a couple of keywords expressing:

  • Mood;
  • Source, for example a Twitch channel or a specific web resource;
  • Writing style, such as ascii, hieroglyphic characters, or another language;
  • What is being discussed in the copy-paste.

It is thanks to tags that it is easier for people to find and become familiar with certain categories of pastes.
