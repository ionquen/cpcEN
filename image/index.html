ASCII image generator from image

Tool description

Using this tool, you can generate an ASCII image with an arbitrary ASCII character width from the image. Each character of the generated image consists of braille dots and spaces.

Options

The color threshold allows you to choose the optimal level at which the important contours of the original image will be involved in the generated ASCII image.

The width parameter sets the thickness of a single character that makes up the final image.

To invert the base color from which the points are generated, click on the "Invert" button.
